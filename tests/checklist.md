# Kalite Kontrol Listesi
## HTML Validasyonu
- [ ] HTML kodları valide edilmelidir.
- [ ] HTML etiketleri doğru şekilde kullanılmaktadır.
## CSS Uyumlulugu
- [ ] CSS kodları valide edilmelidir.
- [ ] CSS stilleri doğru şekilde uygulanmaktadır.
## JS Hatalari
- [ ] JS kodlarında hata yoktur.
- [ ] JS fonksiyonları doğru şekilde çalışmaktadır.
## Responsive
- [ ] Ana sayfa mobil cihazlarda doğru şekilde görüntülenir.
- [ ] Ana sayfa tablet cihazlarda doğru şekilde görüntülenir.
- [ ] Ana sayfa masaüstü cihazlarda doğru şekilde görüntülenir.
## SEO
- [ ] Meta tagları doğru şekilde kullanılmaktadır.
- [ ] Heading hierarchy doğru şekilde uygulanmaktadır.
- [ ] Alt textler doğru şekilde kullanılmaktadır.
## Guvenlik
- [ ] XSS saldırılarına karşı korunmaktadır.
- [ ] CSRF saldırılarına karşı korunmaktadır.
## Performans
- [ ] Resim boyutları optimize edilmiştir.
- [ ] Kodlar minification uygulanmıştır.
## Erisilebilirlik
- [ ] WCAG 2.1 AA standardına uygunluk sağlanmaktadır.
